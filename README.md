# Projeto de Estudo: .NET 

Descrição
----------
Projeto de estudo sobre desenvolvimento em .NET que implementa pequenos sistemas.
Objetivo: praticar conceitos de orientação a objetos, I/O de console e estruturação de projeto em C#.

Tecnologias
----------
- .NET SDK (versão compatível com o seu ambiente)
- C#
- Pode ser aberto no Visual Studio ou executado com CLI do .NET


Como executar
----------
1. Instale o .NET SDK compatível com o projeto.
2. Pelo terminal, na raiz do repositório:
   - `dotnet build`
   - `dotnet run --project ./Financeiro`
3. Ou abra a pasta no Visual Studio e pressione __F5__ para depurar/executar.

Notas
----------
- Projeto destinado apenas a estudo e aprendizado; contém códigos de exemplo e fluxo simples de console.
- Ajuste a versão do SDK conforme necessário no arquivo de projeto (`.csproj`).
